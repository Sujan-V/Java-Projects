                                                          Number Guessing Game

Description
This Java program is a simple number guessing game where the user tries to guess a
randomly generated number within a specified number of attempts. 
The game provides feedback on whether the guessed number is higher or lower than the target number.

Features
- User-defined number of attempts.
- Randomly generated target number between 1 and 100.
- Feedback on each guess to help the user adjust their next guess.
- Notification of success or failure after all attempts.

Getting Started

1. Compile the Program:
   Open your terminal or command prompt and navigate to the directory containing `Guessnumber.java`. 
   Compile the program using:   javac Guessnumber.java

2. Run the Program
   Execute the compiled Java program with:   java  Guessnumber.java
   
3. Play the Game
   - Enter the number of attempts you want to have.
   - Input your guesses as prompted.
   - Receive feedback on whether your guess is too high or too low.
